Unicorn
===========

### HardWare

`Computer` is Raspberry pi3 B

`GPS` is NEO-M8N

`Compass` is [msensor-HCM365](http://www.msensor.com.cn)

`MCU` is STM32

`FC` is Pixhawk and ACE-ONE

`Vehicle` is HEX4,HEX6,Copter-450L,Copter-550,Copter-25B
```
